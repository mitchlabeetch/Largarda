## 四、角色层（Layer 4 — z=40）

角色使用 DragonBones 骨骼动画，以下是骨骼拆件（一套骨架，多职位换装皮肤）。

> 基础网格：16×32px 像素格（渲染 3x = 48×96px）

| 眼镜（可选） | `slot_glasses`         | 16×8px       | 仅部分职位有，可为空 slot                            |

## 七、资产制作优先级汇总

### P0 必须（美术开工最小集）

1. `bg_room` — 房间背景整张（含地板/墙/窗框）
2. `window_scene_night` — 窗外夜景（参考图当前时段，先做一张）
3. `furniture_monitor` + `furniture_keyboard` + `furniture_desk` — 电脑工作区三件套
4. `furniture_chair` — 椅子
5. `furniture_desk_lamp` + `effect_lamp_cone` — 台灯 + 光晕
6. `effect_monitor_glow` + `effect_monitor_cast` — 显示器发光效果
7. `furniture_bookshelf` + `furniture_bookshelf_books` — 书架（记忆入口 P0）
8. DragonBones 骨骼资产（所有 slot 贴图 × 1职位先做）+ 8个动作
9. `avatar_[job]_static` × 1职位 — 主场景卡片立绘

### P1 增强

- 床三件套（bed_frame / bed_quilt / bed_pillow）
- 地毯
- 窗外其余三张景色（日/傍晚/深夜）
- 植物盆栽
- 挂画
- 其余职位 DragonBones 换装 slot 贴图（×4）
- 其余职位静态立绘（×4）

### P2 装饰

- 地板散落物
- 桌面小摆件
- 台历/笔筒
- 植物摇晃动效
- 随机 idle 变体动效

## 八、自绘规范参考

- **画布尺寸**：所有元素在 1x 尺寸下绘制（如 16×16px），PixiJS 渲染时 scale 3x
- **调色板**：优先使用 design-final.md 中的 `--vo-*` 色彩变量对应值
- **关键颜色**：
  - 木地板：`#8B6340`
  - 墙面暖色：`#6B4A2A`
  - 电脑屏幕青光：`#00FFCC`
  - 台灯暖黄：`#F5A623`
  - 深夜窗外：`rgba(30,50,100,0.18)` 叠加深蓝紫底色
- **像素风格**：无抗锯齿，硬边像素，渲染时必须设置 `PIXI.SCALE_MODES.NEAREST`
- **导出格式**：PNG，透明背景，`image-rendering: pixelated`
